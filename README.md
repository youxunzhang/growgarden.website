# 🌱 种植花园价值计算器

一个专为种植类游戏玩家设计的价值计算工具，帮助玩家优化种植策略，最大化游戏收益。

## ✨ 主要功能

### 🧮 价值计算器
- **精确计算**: 基于植物基础价值、品质加成、季节加成进行精确计算
- **利润分析**: 显示种子成本、净利润和利润率
- **实时更新**: 输入变化时自动重新计算
- **历史记录**: 自动保存最近10次计算记录

### 🌿 植物图鉴
- **完整数据库**: 包含15种常见植物，涵盖蔬菜、水果、花卉、草药
- **详细信息**: 显示基础价值、种子成本、生长时间、利润率等
- **智能搜索**: 支持按名称和描述搜索
- **分类过滤**: 按植物类别筛选

### 📚 种植指南
- **新手入门**: 基础种植技巧和建议
- **经济策略**: 高价值作物推荐和投资建议
- **高级技巧**: 肥料使用、轮作、温室等进阶技巧

### 🎨 用户界面
- **响应式设计**: 完美适配桌面、平板、手机
- **现代化UI**: 美观的渐变背景和卡片设计
- **流畅动画**: 平滑的过渡效果和交互动画
- **导航系统**: 固定导航栏，支持平滑滚动

## 📖 使用指南

### 价值计算
1. 在"价值计算器"部分选择植物

## 🎯 计算规则

### 基础价值
```
基础价值 = 植物基础价值 × 数量
```

### 品质加成
- 普通: 1x (无加成)
- 良好: 1.5x (+50%)
- 优秀: 2x (+100%)
- 完美: 3x (+200%)

### 季节加成
- 无加成: 1x
- 春季/夏季/秋季/冬季: 1.2x (+20%)

### 总价值计算
```
总价值 = 基础价值 + 品质加成 + 季节加成
净利润 = 总价值 - 种子成本
利润率 = (净利润 / 种子成本) × 100%
```

## 🤝 贡献指南

## 🕹️ 游戏信息采集脚本

仓库提供了 `scrape_gamedistribution.py` 用于批量抓取 [gamedistribution.com](https://gamedistribution.com/) 的游戏详情信息并下载图标。

### 使用方法

```bash
python scrape_gamedistribution.py \
  --urls https://gamedistribution.com/games/hawaii-match-5/ \
  --output data/games.json \
  --img-dir img
```

- `--urls` 可以同时传入多个游戏链接或游戏别名（slug）。
- `--input-file` 参数允许从文本文件读取更多链接，每行一个，支持 `#` 注释。
- `--output` 指定保存 JSON 数据的位置（默认 `data/games.json`）。
- `--img-dir` 指定图标保存目录（默认 `img/`）。

脚本会自动合并已有的 JSON 数据并在抓取成功时更新/新增条目，同时将图标保存到指定目录下。
